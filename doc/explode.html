Method explode

Splits a string on comma, or a parameter, and trims each value in the resulting Array

Parameters:

  • obj must be a String.
    (String to capitalize)

  • arg must be a String.
    (String to split on)

Returns an Array
(Array of the exploded String)

function explode ( obj, arg ) { arg = arg || ","; return obj.replace( /^(\s+|\t+)|(\s+|\t+)$/g, "" ).split( new RegExp( "\\s*" + arg + "\\s*" ) ); }
